# Personal Finance CLI Assistant (Python)

## Cel projektu
Prosty system śledzenia wydatków w terminalu.  
Ćwiczenie fundamentów Pythona: input, listy, funkcje, pliki, pętle.

## Funkcjonalności
- Dodawanie wydatków (kwota, kategoria, data)
- Wyświetlanie wydatków
- Sumowanie
- Zapis i odczyt z pliku

## Plan rozwoju
- Walidacja danych wejściowych
- Statystyki miesiąca
- Eksport do CSV
- Wersja GUI lub web
